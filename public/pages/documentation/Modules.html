Nodemon

Nodemon is a very nice tool for developing. It wraps your application and listens for changes in your code. If you dont specify which files Nodemon should look for changes in, the default will be js, mjs and json file extensions. You can create a nodemon.json file where you can configure what nodemon should listen for, like in the example below.

nodemon.json

    
        {
            "ext": "js, html, json"
        }

Dev dependencies

global and local
